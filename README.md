# ✨ Sitio-web-de-recetas
Aplicación web para buscar y filtrar recetas de forma rápida y sencilla.

# ✨ Criterios de aceptación (frontendmentor)
Consulta las páginas de inicio, acerca de, índice de recetas y detalles de recetas
Busca recetas por nombre o ingrediente
Filtra las recetas por tiempo máximo de preparación o cocción
Consulta la disposición óptima de la interfaz según el tamaño de pantalla del dispositivo
Consulta los estados de paso del cursor y de enfoque para todos los elementos interactivos de la página
